# Artwork Gallery

A simple data table for browsing artworks from the Art Institute of Chicago.

## Getting Started

```bash
npm install
npm run dev
```

## What it does

- Displays artwork data in a paginated table
- Select rows across multiple pages
- Server-side pagination for better performance

## Stack

React + TypeScript + Vite  
PrimeReact for the data table  
Art Institute of Chicago API
